Interactive Visualisation: Bubble chart using d3.js

Brief description

In this bubble chart, each course is drawn as a circle on a two or multiple dimensional plot. Each course in the database has the following features: Mode, Level, Sector, Price, Duration, Langue. These features will be represented through different dimensions in the visualisation. Clearly the visualisation with all the sectors is too cluttered and is un-informative. However on selecting a particular sector, only those courses belonging to the sector will be shown thereby making it easy to use.

X-axis: Mode

The feature Mode can take one of the values: Distance, Continu, Cours du soir, Cours dâ€™emploi, Partiellement presentiel, and forms the X-axis of the plot.

Y-axis: Level

The feature Level can take one of the values: Advance, Cours specialises, Formation profession- nelle superieure, Sport-etudes, and forms the Y-axis of the plot.

Bubble size: Price

The radius of the bubble is a function of the price of the course.

Mouseover: Course information

The other features such as Duration, Langue, Course name will be shown when the mouse is hovered over a particular bubble.

Pulsating bubbles: Trending courses

Based on web scraping, a few courses in the database was found to be trending or popular. These courses will be shown as pulsating bubbles in the plot.
